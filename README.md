Travel Agency Guide System (UTICA 2003)

An advanced Excel model for automating travel guide payments and region-based bonuses.

Key Functionalities :
- Honorarium Calc: Computes basic fees based on (Days Worked × Daily Rate of 30.000 DT).
- Conditional Bonuses:
- Bonus 1: 20% increase for trips to Tozeur* or Matmata.
- Bonus 2: 5% extra for Tozeur visits exceeding 5 days.
- Reporting: Filtered lists for specific travel destinations (e.g., Tozeur).

 Usage Guide:
1. Fill guide names, visit regions, and entry/exit dates.
2. Apply logical IF formulas for Prime 1 and Prime 2.
3. Save and view the total honorarium for each mission.

Compliant with UTICA 2003 Standards*
